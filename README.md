# Mini Ecommerce - NodeJS

Mini ecommerce app using NodeJS, it has products, shopping cart, manage products and categories

## Installation

Use the package manager [npm] to install.

```bash
npm install --save
```

## Features

* Listing products
* Manage products
* Manage categories
* Add to shopping cart
* View and clear shopping cart

## Used Packages

* Express
* KrakenJS
* DustJS
* MongoDB
* Mongoose
